    Utilisation du module Audience pour suivre les visionneuses

    Dans cette rubrique, vous apprendrez comment le module Audience peut être utilisé pour suivre l'activité de la visionneuse.

    Le module Audience permet de suivre l'activité de visualisation. Dans ce scénario, l'affichage des données n'est pas envoyé à une plate-forme d'automatisation marketing. Un cas d'utilisation courant pour cela serait les vidéos de formation internes. Le suivi de l'audience peut être utilisé pour suivre l'activité de visualisation des employés internes. Pour utiliser le suivi de l'audience, les étapes suivantes sont requises :

    1. Configurer les paramètres de suivi de l'audience
    2. Configurer un formulaire de lecteur et de prospecte pour le suivi de l'audience
    3. Afficher les données vidéo

    Configuration des paramètres de suivi de l'audience

    Pour configurer les paramètres de suivi de l'audience, procédez comme suit :

    1. Ouvrez le module Audience, puis cliquez sur Connexions de données dans la navigation de gauche.
    2. Recherchez la section SUIVI DE L'AUDIENCE , puis cliquez sur Afficher.
    3. Par défaut, le suivi de l'audience est activé pour tous les comptes. Modifiez les paramètres selon vos besoins :
      • Connection Name - Identifie la connexion
      • État de la connexion : détermine si les données d'engagement vidéo Video Cloud sont enregistrées par le module Audience.
        • Activé - Les données sont collectées
        • Désactivé - Aucune donnée n'est collectée (la connexion est désactivée)
      • Suivi des données pour les visionneuses inconnues : détermine si les données de suivi sont enregistrées pour les visionneuses inconnues. Pour identifier les spectateurs lors de l'utilisation du suivi d'audience, un formulaire de prospect d'audience peut être utilisé.
        Remarque : Avec le suivi de l'audience, jusqu'à 1 000 spectateurs seront suivis par période de 24 heures.
      • Combien de temps Brightcove doit-il conserver les données ? - La politique de conservation des données détermine la durée de conservation des données de visualisation et des données de formulaire de prospect (si vous utilisez un formulaire de prospect Audience) dans votre compte Video Cloud. Vous devez ajuster les paramètres de rétention en fonction de votre politique de confidentialité - il n'est pas nécessaire de conserver les données utilisateur dans Video Cloud.

    Configuration d'un lecteur et d'un formulaire de prospecteur pour le suivi de l'audience

    L'activité de visualisation ne peut être suivie que lorsque les vidéos sont affichées à l'intérieur d'un lecteur vidéo compatible Audience. Vous devez utiliser un lecteur compatible Audience lorsque vous utilisez le suivi d'audience. Les formulaires de prospect peuvent être affichés pendant la lecture vidéo pour capturer les informations du spectateur. Les formulaires de prospect ne sont pas requis lors de l'utilisation du suivi d'audience.

    Pour configurer le suivi d'audience avec un lecteur compatible Audience et un formulaire de prospect, procédez comme suit :

    1. Cliquez sur Audience Lecteurs dans la navigation de gauche.
    2. Déterminez le lecteur compatible Audience à utiliser. Si aucun lecteur compatible Audio n'existe, cliquez sur Créer un lecteur pour en créer un. Pour plus d'informations sur la création de lecteurs compatibles Audience, voir Création d'un lecteur compatible Audience.
    3. Bien qu'un formulaire de prospect n'est pas requis, les formulaires de prospect peuvent être utilisés pour recueillir des informations sur le spectateur et transformer les spectateurs anonymes en spectateurs connus. Cliquez sur Formulaires de prospect et créez un formulaire de prospect. Pour plus d'informations sur la création de formulaires de prospect, voir Création d'un formulaire de prospect d'audience.
    4. Cliquez sur Lecteurs d'audience.
    5. Localisez le lecteur compatible Audience à utiliser. Définissez la valeur Connexion à du lecteur sur Suivi de l'audience.
    6. Si un formulaire prospect est utilisé, sélectionnez-le dans la liste déroulante Formulaire de prospect .

    Affichage des données vidéo

    Plusieurs options sont disponibles pour visualiser des données vidéo lors de l'utilisation du suivi d'audience :

    • Activités du spectateur (Événements) - Fait partie du module Audience, cette option peut être utilisée pour visualiser des données vidéo dans le module Audience. Pour plus d'informations, voir Affichage des événements du lecteur dans le module Audience.
    • Profils de visionneuse - Fait partie du module Audience, cette option peut être utilisée pour visualiser des données vidéo en fonction d'une valeur de profil utilisateur dans le module Audience. Pour plus d'informations, voir Affichage des informations de profil de la visionneuse
    • Exporter les données - Fait partie du module Audience, cette option peut être utilisée pour exporter des données de vue vidéo et de prospect. Pour plus d'informations, voir Exportation de données d'audience à partir du module Audience.
    • API Audience - L'API Audience peut être utilisée pour extraire par programme des données de vue vidéo et de prospect. Pour plus d'informations, consultez Généralités : API d'audience.
