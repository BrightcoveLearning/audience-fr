    Création d'un objet Salesforce personnalisé pour l'intégration Pardot

    Dans cette rubrique, vous apprendrez comment créer l'objet personnalisé Salesforce requis lors de l'utilisation de l'intégration Pardot.

    Pour envoyer des données sur la plateforme Pardot, vous devez envoyer des données d'affichage au niveau contact à votre CRM Salesforce via un objet personnalisé. Vous pouvez ensuite utiliser les fonctionnalités natives de Pardot et Salesforce CRM pour ingérer le nouvel objet personnalisé dans Pardot.

    Exigences

    Pour utiliser l'intégration Salesforce - Pardot, vous devez :

    • Salesforce et Pardot Connector doivent être installés et configurés. Contactez votre responsable de compte Pardot si vous avez besoin d'aide.
    • Vous devez disposer d'un mappage d'objets personnalisés disponible dans votre compte Pardot. Contactez votre gestionnaire de compte Salesforce si vous devez acheter des mappages d'objets supplémentaires.

    Objets Salesforce

    Les objets sont des tables de base de données qui vous permettent de stocker des données spécifiques à l'organisation dans Salesforce. Il existe deux types d'objets dans Salesforce :

    • Objets standard - Les objets fournis par salesforce.com sont appelés objets standard. Exemples d'objets standard : Comptes, Contacts, Opportunités, prospects, produits, campagnes, requêtes, utilisateurs, contrats, rapports et tableaux de bord
    • Objet personnalisé - Les objets créés par les utilisateurs sont appelés objets personnalisés. Les objets personnalisés stockent des informations uniques et importantes pour votre organisation et fournissent une structure de partage des données. Les données de visualisation vidéo peuvent être envoyées à partir de Brightcove Players à Salesforce à l'aide d'objets personnalisés. Différents champs avec différents types de données peuvent être ajoutés à l'objet. Une relation de recherche relie essentiellement deux objets ensemble afin que vous puissiez « rechercher » un objet à partir des éléments associés sur un autre objet. Nous pouvons créer une relation de recherche entre l'objet personnalisé et un contact.

    Création d'un objet personnalisé

    Les contacts sont les utilisateurs dont vous souhaitez suivre ou conserver des enregistrements dans Salesforce. Il doit y avoir une relation entre l'objet personnalisé créé ci-dessous et l'objet Contact dans Salesforce afin d'obtenir des données liées à l'utilisateur correct. Pour créer cet objet personnalisé, procédez comme suit :

    Remarque : Les images d'écran ci-dessous ont été prises à l'aide de l'interface Salesforce Lightning.
    1. Connectez-vous à Salesforce.
    2. Cliquez sur l'icône d'engrenage dans le coin supérieur droit pour ouvrir le menu Configuration .
    3. Dans la navigation de gauche, dans la section OUTILS PLATE-FORME , cliquez sur Gestionnaire > d'objets et champs.
    4. En haut de la page Gestionnaire d'objets , cliquez sur Créer un objet > personnalisé.
    5. Saisissez une étiquette. Brightcove recommande d'utiliser BC Video View.
    6. Saisissez un libellé pluriel. Brightcove recommande d'utiliser BC Video Views.
    7. Saisissez un nom d'objet. Une valeur initiale sera générée en fonction de la valeur Label .
    8. Entrez un nom d'enregistrement. Une valeur initiale sera générée en fonction de la valeur Label . Confirmez que le type de données est Texte.
    9. Sélectionnez toutes les fonctionnalités facultatives.
    10. Cliquez sur Save.
    11. Sur la page Détail de l'objet personnalisé , recherchez la section Champs personnalisés et relations .
    12. Cliquez sur Nouveau.
    13. Créez les champs personnalisés suivants. Pour tous les champs, définissez le Type de données sur Texte et conservez les options de sécurité et de mise en page par défaut au niveau des champs.

    14. Confirmez que tous les champs ont été créés.
    15. Dans la section Champs personnalisés et relations , cliquez sur Nouveau.
    16. Sélectionnez Relation de recherche et cliquez sur Suivant.
    17. Dans la liste déroulante Relé à , sélectionnez Contact. Cliquez sur Next (Suivant).

      Cela associera l'objet personnalisé BC Video View à un contact dans Salesforce.

    18. Pour Nom du champ, saisissez Contact. Cliquez sur Next (Suivant).
    19. Conservez les options de sécurité par défaut au niveau du champ. Cliquez sur Next (Suivant).
    20. Sur la page du champ de référence (étape 5), cliquez sur Suivant.
    21. Sur la page des listes associées personnalisées (étape 6), cliquez sur Enregistrer.

    En règle générale, lorsque vous utilisez le type d'intégration Pardot, vous sélectionnez le type de mappage avancé dans Brightcove Audience, puis mappez les champs d'affichage à l'objet personnalisé Salesforce que vous avez créé.

    Affichage des données dans Pardot

    Une fois les données synchronisées, vous pouvez afficher les objets et les données dans Pardot.

    1. Dans la navigation de gauche, passez la souris sur Admin et cliquez sur Objets personnalisés.
    2. Cliquez sur + Ajouter un objet personnalisé.
    3. Sélectionnez l'objet personnalisé BC Video View .
    4. Dans la navigation de gauche, passez la souris sur Prospects et cliquez sur Liste des prospects. Les contacts créés dans Salesforce seront affichés.
    5. Cliquez sur un nom de prospect.
    6. Cliquez sur l'onglet Objets associés . Vérifiez que les données d'objet personnalisées s'affichent (il s'agit des mêmes données que dans Salesforce).
    7. Cliquez sur le nom de la vidéo pour voir les détails.
