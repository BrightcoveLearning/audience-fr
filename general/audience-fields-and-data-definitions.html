    Champs d'audience et définitions de données

    Cette rubrique fournit une vue d'ensemble des champs qui peuvent être renseignés par le module Audience.

    Brightcove Audience fournira un ensemble de données de visualisation vidéo dans des champs spécifiques de votre plate-forme intégrée. Voici un exemple de ces champs et une description de ce que sont les données qui rempliront chaque champ

    Champs standard

    Ces champs seront toujours remplis par Brightcove Audience.

    Champ Description
    video_id

    ID de la vidéo Brightcove associée à l'événement de visualisation
    nom_video_

    Le nom de la vidéo dans Video Cloud
    tracking_id Vous pouvez ajouter un ID de suivi personnalisé à une URL
    external_id ID utilisateur global (GUID) utilisé pour identifier l'utilisateur
    player_id ID du lecteur Brightcove qui a créé l'événement view
    page_url L'URL de la page où la vidéo a été visionnée
    regardait Pourcentage regardé - Jusqu'à quel point ils ont regardé
    time_watched Secondes de la vidéo regardée - Le temps total qu'ils ont réellement passé avec cette vidéo
    created_at Date de création
    updated_at Date de dernière mise à jour
    is_syncé Un booléen indiquant si l'événement view a été synchronisé ou non avec la plate-forme tierce.

    Champs supplémentaires

    Ces champs ne sont pas renseignés automatiquement. Ils sont utilisés pour suivre des données personnalisées supplémentaires. Par défaut, ils sont configurés pour capturer l'interactivité à partir d' expériences Gallery In-Page. Si vous n'utilisez pas l'interactivité, ces champs seront vides.

    Champ Description
    event_1, event_2, event_3

    Événements personnalisés
    metric_1, metric_2, metric_3

    Mesures personnalisées
    Champs UTM - UtmSource, UTMMedium, UtmCampaign, UtmTerm et UTMContent Pour plus d'informations sur l'utilisation des champs UTM, voir Utilisation des champs UTM dans le module Audience.

    Champs personnalisés

    Ces champs ne sont pas renseignés automatiquement. Vous avez la possibilité de synchroniser l'ID de référence, les balises ou les champs personnalisés avec votre MAP ou CRM via ces champs. Pour le configurer, contactez le support technique et demandez, par exemple : « Veuillez mapper mes « tags » de mon champ Video Cloud au champ Audience « custom_1 ». »

    Champ Description
    custom_1, custom_2, custom_3

    Événements personnalisés
