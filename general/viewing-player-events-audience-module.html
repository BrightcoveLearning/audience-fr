    Affichage des événements de lecteur dans le module Audience

    Dans cette rubrique, vous apprendrez comment afficher les événements des joueurs dans le module Audience.

    Le module Audience permet de suivre les événements de vue. La fréquence à laquelle les événements de vue sont envoyés au module Audience est calculée dynamiquement en fonction de la longueur de la vidéo. Le public enverra au moins 4 événements, un par quartile.

    Remarque : Les données d'événement View peuvent être exportées à l'aide de l'option Exporter les données ou à l'aide de l' API Audience.

    Pour afficher les événements d'affichage, ouvrez le module Audience et cliquez sur Toutes les activités récentes dans la navigation de gauche. La page affichera l'activité de visualisation de l'événement au cours des dernières 24 heures.

    Le lien Afficher les événements d'interaction est activé si des événements ou des mesures personnalisés sont attachés à un événement de vue.

    La page affiche les informations suivantes.

    • Horodatage - Date et heure de la vue vidéo
    • Nom et ID de la vidéo - Nom vidéo et ID vidéo Video Cloud

    • Pourcentage regardé - Le point auquel le spectateur a cessé de regarder la vidéo, divisé par la longueur de la vidéo. Par exemple, si un spectateur a regardé les 5 premières secondes d'une vidéo de deux minutes, puis passe au point 55 secondes, regardait 5 secondes de plus et arrête la vidéo (ou quitte la page), le pourcentage regardé est enregistré comme 50 %.

      Remarque : Pour les vidéos en direct, il n'est pas possible de suivre le pourcentage de visionnage dans Audience. Comme il n'est pas possible de savoir combien de temps il durera, le module Audience ne suivra que le temps total regardé.
    • Temps surveillé - Temps total regardé
    • External ID - Valeur utilisée pour identifier la visionneuse, par exemple le GUID Eloqua, le GUID Marketo, le GUID HubSpot, le GUID SSO ou une autre valeur unique qui a été utilisée pour identifier le spectateur.
    • État - État de l'événement
      • non traité - l'événement n'a pas été synchronisé avec la plateforme d'automatisation marketing
      • synchronisé - l'événement a été synchronisé avec succès avec la plate-forme d'automatisation marketing
      • sync_disabled - Le module Audience a été configuré avec synchronisation désactivée afin que l'événement ait été ignoré
      • échec - Erreur de synchronisation ; l'info-bulle fournit des détails sur l'erreur qui s'est produite
      • unknown_contact - utilisateur inconnu donc l'événement n'a pas été synchronisé
      • N/A - non applicable ; affiché lors de l'utilisation d'intégrations personnalisées, de suivi d'audience ou de Marketo Munchkin

    Les données d'événement peuvent également être filtrées par état à l'aide de la liste déroulante située en haut de la page.

    Affichage des événements d'interaction

    Si des mesures ou des événements personnalisés sont attachés à un événement de vue, le lien Afficher les événements d'interaction est activé. Par exemple, si vous utilisez une expérience Galerie dans la page avec un lecteur compatible Audience et que l'expérience est configurée avec des liens, des cartes ou tout autre type d'élément interactif, l'expérience enverra des événements d'impression et de conversion au module Audience. Ces événements sont transmis à Audience à l'aide des champs personnalisés Audience. Les données de l'événement apparaîtront dans le module Audience dans le cadre de l' activité All Recent.

    Remarque : Les événements d'interaction peuvent également apparaître lors de l'utilisation de mesures et d'événements personnalisés d'audience.

    Les données suivantes peuvent être transmises :

    • Événement 1 - Liste séparée par des virgules des noms de CTA avec lesquels le spectateur a eu la chance d'interagir (impression)
    • Événement 2 - Liste séparée par des virgules des noms CTA avec lesquels le spectateur a interagi (conversion)
    • Mesure 1 - Nombre total d'impressions CTA
    • Mesure 2 - Nombre total de conversions de DEC, c.-à-d. lien cliqué, soumission de formulaire, etc.

    Pour afficher les événements, cliquez sur le lien Afficher les événements d'interaction dans l'en-tête.

    Autres options de visualisation des données

    Les données d'audience peuvent être consultées/exportées d'autres manières :

    • Activité vidéo - Permet d'afficher l'activité du spectateur par vidéo. Pour plus d'informations, voir Affichage de l'activité vidéo.
    • Visualiseur Profils - Utilisé pour afficher l'activité vidéo par visionneuse. Pour plus d'informations, voir Affichage des informations de profil de la visionneuse
    • Exporter des données : permet d'exporter des données de vue vidéo et de prospect. Pour plus d'informations, voir Exportation de données d'audience à partir du module Audience.
    • API Audience - L'API Audience peut être utilisée pour extraire par programme des données de vue vidéo et de prospect. Pour plus d'informations, consultez Généralités : API d'audience.
